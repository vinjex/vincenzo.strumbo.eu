# vincenzo.strumbo.eu

### Prerequisites
#### Install required Node packages
    $ npm i
___
### Developing
    # Serve and watches for files modification
    $ npm run-script serve
___
### Building
    # Production code will be saved to dist/
    $ npm run-script build
___
